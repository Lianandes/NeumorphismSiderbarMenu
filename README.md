# Neumorphism Sidebar Menu

Projeto desenvolvido para praticar a criação de interfaces modernas utilizando HTML e CSS através de um menu lateral no estilo neumorphism.

## 🚀 Objetivo

O objetivo deste projeto foi aprimorar conhecimentos em design UI, estilização avançada com CSS e construção de componentes modernos inspirados no conceito de neumorphism.

## 🛠 Tecnologias utilizadas

- HTML5
- CSS3

## ✨ Funcionalidades

- Sidebar menu interativo
- Design neumorphism
- Interface moderna
- Efeitos de profundidade
- Layout organizado

## 📚 Conceitos praticados

- Neumorphism Design
- Box Shadow
- Flexbox
- Estilização moderna
- Estruturação de interface
